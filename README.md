# Zenex Knowledge Hub — Design Prototype

A working prototype built by **Veritech Digital** for the Zenex Foundation panel presentation (27 July 2026).

**Live demo:** https://veri-tech.github.io/zenex-demo-site/

## What this is
A stakeholder-centred redesign concept for the Zenex Foundation Knowledge Hub — built on the real site's branding, photography and structure, with a proposed audience-adaptive publication system layered in.

## Pages
- `index.html` — Homepage (real hero carousel, Strategy 2030, Blog Posts, 30 Years of Impact, Contact Us)
- `knowledge-hub.html` — Knowledge Hub landing page
- `search.html` — Filtered search with live facet filtering
- `publication.html` — Publication page with a live Government/NGO/Donor/Researcher view switcher

## Notes
- Fully self-contained — no build step, no server required. Open `index.html` directly, or serve statically.
- All images are base64-embedded in `assets.js` (sourced from the real Zenex Foundation site).
- Not the live Zenex Foundation website — a design prototype only.
